# media_control.py
- Run the program
  - The imports may take some time
- With your right hand make a gesture inside of the green bounding box to control your media player (works best with a neutral background)
  - Stop = play/pause
  - Thumbs up = volume up
  - Thumbs down = volume down
- Press Q to exit
